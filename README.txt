В случае кодирования (сжимаем оригинальный файл и шифруем его), в .bat файл заносятся следующие аргументы:
имя_исполнительного_приложения e_(то есть кодирование) имя_сжимаемого_файла БУДУЩЕЕ_имя_сжатого_файла ключ_шифрования
На выходе имеем сжатый файл и зашифрованный сжатый файл.

В случае декодирования (расшифровываем модифицированный файл и распаковываем его), в .bat файл заносятся следующие аргументы:
имя_исполнительного_приложения d_(то есть декодирование) имя_зашифрованного_файла имя_сжатого_файла БУДУЩЕЕ_имя_восстановленного_файла ключ_шифрования
На выходе имеем разархивированный распакованный файл, идентичный оригинальному.

Экспериментируя, можно сделать некоторые выводы о сжимаемости разного типа информации. Алгоритм LZW имеет потенциал для оптимизации и повышения эффективности сжатия.
